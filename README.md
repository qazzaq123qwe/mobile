## Описание
**CryptoPulse** — это мобильное Android-приложение для мониторинга рынка криптовалют в реальном времени и управления персональными цифровыми активами. Проект разработан с использованием актуальных компонентов Android Jetpack и демонстрирует реализацию архитектурного паттерна MVVM, работу с локальной базой данных Room и сетевыми запросами к CoinGecko API.


---

## Технический стек
* **Язык разработки:** Java 
* **Минимальная версия SDK:** 24 (Android 7.0 Nougat)
* **Целевая версия SDK:** 34 (Android 14)
* **Система сборки:** Gradle 9.2.1
* **Архитектурный паттерн:** MVVM (Model-View-ViewModel)
* **Сетевой слой:** Retrofit 2, OkHttp 3 (логирование запросов, кастомные интерцепторы)
* **Локальная БД:** Room Persistence Library (кэширование данных, хранение транзакций)
* **Компоненты UI:** ViewBinding, Material Design 3, RecyclerView (с оптимизацией через DiffUtil), SwipeRefreshLayout

---

## Детальный функционал

### 1. Мониторинг рынка (`ui.main`)
* **Сетевые запросы:** Асинхронное получение списка ТОП-монет с сортировкой по рыночной капитализации.
* **Обновление данных:** Поддержка жеста Swipe-to-Refresh для принудительного обновления котировок.
* **Поиск:** Фильтрация списка в реальном времени по названию криптовалюты или её тикеру (например, BTC, ETH) без повторных запросов к сети.

### 2. Управление портфелем (`ui.portfolio`)
* **Локальное хранилище:** Все добавленные транзакции (покупка/продажа монет) сохраняются локально на устройстве в SQLite через Room.
* **Расчет PnL:** Приложение сопоставляет историческую цену покупки актива с его текущей рыночной стоимостью и выводит общий баланс кошелька, а также процент чистой прибыли или убытка.
* **Реактивный интерфейс:** Изменения в базе данных автоматически обновляют UI благодаря связке LiveData.

### 3. Локализация и доступность
* Ресурсы строк полностью вынесены в файлы `strings.xml`.
* Добавлена встроенная поддержка нескольких языковых локалей (английский, русский, немецкий, испанский, украинский). Переключение происходит автоматически на основе системных настроек устройства.
